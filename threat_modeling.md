## Threat Modeling Flowchart

```mermaid
flowchart LR
    subgraph Scope
        A[Assets / System / Scope]
    end
    subgraph Identify
        B[Threats / Attack Vectors]
    end
    subgraph Assess
        C[Likelihood / Impact]
    end
    subgraph Mitigate
        D[Controls / Priorities]
    end
    A --> B --> C --> D
```
